# Анализ датасета Skyrim Weapons

Данные взяты с Kaggle https://www.kaggle.com/datasets/elmartini/skyrim-weapons-dataset и содержат информацию о различных типах оружия в игре The Elder Scrolls V: Skyrim, их стоимости и уроне

В данной работе проведен разведывательный анализ данных, обработаны пропуски и выбросы, созданы новые признаки (DPS), проверено несколько гипотез и реализована модель машинного обучения, предсказывающая урон оружия в зависимости от других хакратеристик

## Новые признаки
Для лучшей интерпретации урона был создан признак "DPS" - Damage Per Second (урон в секунду). Для этого понадобились данные о скорости ударов различных типов оружия. они были загружены с сайта Unofficial Elder Scrolls Pages. Скорость для луков и арбалетов рассчитывалась по определенной формуле с этого сайта.

## Гипотезы
Гипотезами проверены 
- самые распространенные типы оружия в игре
- самые распространенные категории
- самые дорогие материалы
- лучший по урону и DPS тип оружия
- лучший по "выгоде" тип оружия (топ за свои деньги)
- зависимость урона оружия от его веса

## Модель линейной регрессии

Использованы количественные переменные и качественные, закодированные с помощью метода One-Hot-Encoding, R^2 модели составил 0.892
